# CodeAlpha Ecommerce Store

## Overview

A Full Stack Ecommerce Web Application developed using React.js, Node.js, Express.js, and MongoDB.

## Features

* User Registration
* User Login & Logout
* Product Listing
* Search Products
* Category Filtering
* Shopping Cart
* Checkout System
* Order History
* MongoDB Integration

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

## Installation

### Frontend

npm install

npm run dev

### Backend

npm install

npm start
